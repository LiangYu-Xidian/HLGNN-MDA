HLGNN-MDA 
===============================================================================

About
-----

A miRNA-disease association prediction approach HLGNN-MDA


Installation
------------

Install [PyTorch](https://pytorch.org/).

Type

    bash ./install.sh


Usages
------

Type:

    python Main.py.py --bi-network --first-kind-node-num 0 --hop 1 --seed 1

to run HLGNN-MDA with hop 1 subgraph
